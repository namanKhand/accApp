# UI & Screen Inventory

## Core Screens
- **Onboarding carousel (4 slides)**: concept intro and partner responsibility messaging.
- **Phone login**: phone input, OTP entry, error handling, resend timer.
- **Profile setup**: name, photo, timezone selection.
- **Home dashboard**: goals list, todayâ€™s status, partner status, quick nudge, streak snippet.
- **Goal setup (mad-lib)**: form with live preview, reminder times, partner selection.

- **Daily check-in**: camera/gallery picker, photo preview, note input, submit.
- **Streak view**: bar + calendar with completed/missed markers, longest/current streak.

- **Profile/settings**: profile edit, notifications, partner management, logout.

## Components
- Buttons (primary, secondary, success, disabled) using fall-inspired palette.
- Cards for goals and check-ins.
- Toasts/alerts for success, nudges, and errors.
- Progress indicators for uploads and pending actions.
