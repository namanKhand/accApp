# Feature Scope

## MVP Features
- **Goal creation**: structured template with title, frequency, deadline, partner selection, and reminders.
- **Accountability partner invite**: phone/email invite with status (pending/accepted) and partner responsibility messaging.
- **Daily check-ins**: photo or gallery upload, timestamp, optional notes, automatic streak update.
- **Streaks**: daily streak calculation with reset on missed check-in; visual streak bar/calendar.
- **Nudges**: partner-triggered reminders when a check-in is missed; push notifications to user.
- **Home dashboard**: user goal cards, todayâ€™s check-in state, partner status, quick nudge button.
- **Profile/Settings**: profile info, notification preferences, partner management, logout.

## Near-Term (V2)
- In-app messaging and reactions on check-ins.
- Analytics and insights (completion rates, time-of-day heatmaps).
- Group goals or small accountability circles.
- Habit templates and suggestions.
- Calendar sync for milestones.
- Web companion dashboard.

## Non-Goals (Now)
- Full gamification system.
- Public social feed.
- Extensive rewards marketplace.
